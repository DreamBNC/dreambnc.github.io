We, at DreamBNC, use open source software, every day. If you know it or not, you do too. For example:
  • Chrome is based off an open source browser, Chromium.
  • Firefox is completely open source.
  • WordPress is open source.
So we, at DreamBNC, couldn't turn our heads away. ZNC is open source too! We made a fork of ZNC. It was named DreamZNC. So, are you interested? Want to dig in some code or fix a spelling mistake? Come and contribute. You can find DreamZNC source at our GitHub repository. And hey, don't work alone. Do you want help from the DreamZNC developers? Do you want to give them useful tips? Join #DreamBNC-dev at Rizon and let's start some interesting discussions.
